# pickles

The perfect picker for parsing perplexing JSON properties.

## Features

- **JSON Parsing**: Easily upload and parse local JSON files.
- **Flattened View**: Automatically flattens nested JSON structures into an easy-to-read list of paths and values.
- **Smart Search**: 
  - Fuzzy search capabilities across both keys/paths and values.
  - Autocomplete suggestions as you type.
- **Results Display**:
  - **Primitives**: Sortable table view for primitive values (strings, numbers, booleans). Sort by Value or Path.
  - **Objects**: Dedicated view for object structures with sorting options (Insertion Order, A-Z, Z-A).
- **Clipboard Integration**: 
  - One-click copying of JSON paths.
  - **"Copy last property only"** mode to grab just the leaf key instead of the full path.
- **Visual Helpers**: 
  - Automatic color previews for hex codes.
  - Dark mode support for comfortable viewing.
- **Persistence**: Automatically saves your loaded file, search state, and preferences locally, so you can pick up right where you left off.
